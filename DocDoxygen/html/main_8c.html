Detailed Description

Основной файл программы, включающий в себя функции инициализации периферии, основной цикл программы и обработку данных.

Этот файл содержит в себе основной цикл программы, инициализацию всех необходимых периферийных устройств и обработку данных, получаемых от акселерометра ADXL362 и передачу данных по UART.

void CreateTxBuff ( uint8_t * xb,

Собирает буфер для передачи данных по USART.

Note
На вход приходят 3 переменные (значения регистров XDATA, YDATA, ZDATA). На выходе получаем регистр размером UART_TX_BUFFER_SIZE*3+2
Parameters
[in]xbУказатель на буфер с данными XDATA.
[in]ybУказатель на буфер с данными YDATA.
[in]zbУказатель на буфер с данными ZDATA.
[in,out]tbУказатель на буфер для передачи данных.
